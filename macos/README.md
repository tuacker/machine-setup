# macOS setup

Bootstrap script for a fresh macOS install. Installs core tools/apps, configures shell + dev tooling, applies preferred macOS defaults, and prints manual follow-ups at the end.

## Run

```bash
./macos/setup.sh
```

## Options

- `--global-only`
- `--user-only`
- `--defaults-only`
- `--skip-global`
- `--skip-user`
- `--skip-defaults`
